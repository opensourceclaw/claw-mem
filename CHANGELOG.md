## [0.1.0] - 2026-03-17

### ✨ Added

- **Three-Layer Memory Architecture**
  - L1: Working Memory
  - L2: Short-term Memory
  - L3: Long-term Memory
  - Memory transfer strategy

- **Self-Baking Pipeline**
  - Raw → Tagged → Summarized → Structured → Consolidated
  - Automatic tag generation
  - Intelligent summary compression

- **Basic Retrieval**
  - Keyword search
  - N-gram hash lookup
  - Category filtering

- **Session Management**
  - Automatic session lifecycle management
  - Auto memory transfer on session end
  - Session context restoration

## Version Notes

### Version Numbering

- **MAJOR** - Incompatible API changes
- **MINOR** - Backward compatible new features
- **PATCH** - Backward compatible bug fixes

### Release Cycle

- **PATCH** - As needed (bug fixes)
- **MINOR** - Monthly (new features)
- **MAJOR** - Quarterly (major updates)

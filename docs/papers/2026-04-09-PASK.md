## 核心贡献

> "Proactivity 是 AGI 的核心期望.现有工作仍主要局限于实验室设置,在真实世界主动 Agent 方面存在明显差距."

**DD-MM-PAS 范式**:
- **DD (Demand Detection)**: 需求检测
- **MM (Memory Modeling)**: 记忆建模
- **PAS (Proactive Agent System)**: 主动 Agent 系统

---

## 问题: 被动 AI 的局限

### "天时,地利,人和" 挑战

| 挑战 | 描述 |
|------|------|
| **天时** | 严格时间约束下无法停止查询 AI |
| **地利** | 会议/社交场合调用 AI 尴尬或干扰 |
| **人和** | 使用 AI 需要大量努力:识别意图,构造提示,适应交互风格 |

**核心问题**:
> "AI 必须超越被动查询-响应交互,建立在共享人类感知体验的基础上."

## Pask 实现

### 1. IntentFlow (DD)

**流式结构基础模型**:
- 实时需求检测
- 低延迟约束下匹配 Gemini3-Flash
- 识别更深层用户意图

### 2. 混合记忆 (MM)

**三层记忆架构**:
```
┌─────────────────────────────────────────────────────────────┐
│  Workspace Memory (工作空间)                                 │
│  ├── 当前会话上下文                                          │
│  └── 任务相关信息                                            │
│                                                             │
│  User Memory (用户记忆)                                      │
│  ├── 个人偏好                                                │
│  ├── 历史交互                                                │
│  └── 行为模式                                                │
│                                                             │
│  Global Memory (全局记忆)                                    │
│  ├── 通用知识                                                │
│  └── 跨用户洞察                                              │
└─────────────────────────────────────────────────────────────┘
```

### 3. 自进化记忆

**机制**:
- 记忆自我进化与维护
- 支持长期人-AI 共同进化
- 超越传统聊天机器人系统

### 三层记忆实现

```python
# claw-mem v2.1.0
class PaskStyleMemory:
    """Pask 风格三层记忆"""
    
    def __init__(self):
        # Workspace Memory
        self.workspace = WorkspaceMemory()
        
        # User Memory
        self.user = UserMemory()
        
        # Global Memory
        self.global_mem = GlobalMemory()
    
    def store(self, item: MemoryItem):
        """存储记忆"""
        if item.scope == "workspace":
            self.workspace.store(item)
        elif item.scope == "user":
            self.user.store(item)
        else:
            self.global_mem.store(item)
    
    def retrieve(self, query: str, scope: str = "all"):
        """检索记忆"""
        results = []
        
        if scope in ["all", "workspace"]:
            results.extend(self.workspace.retrieve(query))
        
        if scope in ["all", "user"]:
            results.extend(self.user.retrieve(query))
        
        if scope in ["all", "global"]:
            results.extend(self.global_mem.retrieve(query))
        
        return self._rank_and_merge(results)
    
    def self_evolve(self):
        """自进化"""
        # 压缩,去重,重组织
        self.workspace.compress()
        self.user.update_patterns()
        self.global_mem.aggregate_insights()
```

## 对 neoclaw 的启发

### 主动 Agent 系统

```python
# neoclaw v2.1.0
class ProactiveAgentSystem:
    """主动 Agent 系统"""
    
    def __init__(self):
        self.intent_flow = IntentFlow()
        self.memory = PaskStyleMemory()
        self.always_on = True
    
    def run(self):
        """Always-on 执行循环"""
        while self.always_on:
            # 1. 感知实时信号
            signals = self._perceive()
            
            # 2. 需求检测 (DD)
            latent_needs = self.intent_flow.detect(signals, self.memory)
            
            # 3. 如果有潜在需求,主动帮助
            if latent_needs:
                for need in latent_needs:
                    response = self._execute(need)
                    self._deliver(response)
            
            # 4. 更新记忆 (MM)
            self.memory.self_evolve()
```

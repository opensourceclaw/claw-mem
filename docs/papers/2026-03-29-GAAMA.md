## 问题: 现有方法的局限

### Flat RAG

- 仅通过嵌入相似度检索
- 丢失结构关系
- 无法捕获多会话关联结构

### HippoRAG

- 实体中心设计导致上下文丢失
- 高度节点 (mega-hubs) 稀释 PPR 精度
- 作者承认 "a critical flaw"

### A-Mem / Nemori

- 主要依赖嵌入相似度
- 缺乏图介导的关联路径

## GAAMA 架构

```
┌─────────────────────────────────────────────────────────────┐
│                    GAAMA 知识图谱                            │
│                                                             │
│  四种节点类型:                                               │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  Episode (对话轮次)                                     │  │
│  │  • 逐字保存原始对话                                     │  │
│  │  • 保留时间引用 ("昨天", "上周")                         │  │
│  │  • NEXT 边链接时序                                      │  │
│  │                                                       │  │
│  │  Fact (原子事实)                                        │  │
│  │  • LLM 提取的原子断言                                   │  │
│  │  • DERIVED_FROM 边指向来源                              │  │
│  │                                                       │  │
│  │  Reflection (反思)                                      │  │
│  │  • 跨多个事实的高阶洞察                                  │  │
│  │  • SYNTHESIZED_FROM 边                                  │  │
│  │                                                       │  │
│  │  Concept (概念)                                         │  │
│  │  • 主题级标签 (pottery_hobby, camping_trip)             │  │
│  │  • 提供跨切遍历路径                                      │  │
│  │  • 避免实体中心的 mega-hub 问题                          │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                             │
│  五种边类型:                                                 │
│  NEXT, DERIVED_FROM, SYNTHESIZED_FROM,                     │
│  RELATED_TO, HAS_CONCEPT                                    │
└─────────────────────────────────────────────────────────────┘
```

## 三步构建管道

### Step 1: Episode Preservation (无 LLM)

- 逐字保存对话轮次
- NEXT 边链接时序
- 线性时间复杂度

### Step 2: Fact & Concept Extraction (LLM)

- LLM 提取原子事实
- LLM 提取概念标签
- DERIVED_FROM 边保留来源

### Step 3: Reflection Synthesis (LLM)

- 从多个事实合成高阶反思
- SYNTHESIZED_FROM 边

---

## 检索机制

### Hybrid Scoring

```
score = α × semantic_similarity + β × PPR_score

其中:
- α: 语义相似度权重
- β: PPR 权重 (0.1)
- PPR: Edge-type-aware Personalized PageRank
```

### Hub Dampening

- 解决 mega-hub 问题
- 避免高度节点稀释精度

### GAAMA 风格记忆

```python
# claw-mem v2.1.0
class GAAMAMemory:
    """GAAMA 风格图增强记忆"""
    
    def __init__(self):
        self.graph = KnowledgeGraph()
        self.embedding_store = VectorStore()
        
        # 节点类型
        self.episodes = NodeStore("episode")
        self.facts = NodeStore("fact")
        self.reflections = NodeStore("reflection")
        self.concepts = NodeStore("concept")
    
    def add_conversation(self, turns):
        """添加对话"""
        # Step 1: Episode preservation
        for i, turn in enumerate(turns):
            episode = self.episodes.add(turn)
            if i > 0:
                self.graph.add_edge(
                    turns[i-1], turn, 
                    type="NEXT"
                )
        
        # Step 2: Fact & Concept extraction
        facts = self._extract_facts(turns)
        concepts = self._extract_concepts(turns)
        
        for fact in facts:
            self.facts.add(fact)
            self.graph.add_edge(
                fact, fact.source, 
                type="DERIVED_FROM"
            )
        
        # Step 3: Reflection synthesis
        reflections = self._synthesize_reflections(facts)
        for refl in reflections:
            self.reflections.add(reffl)
    
    def retrieve(self, query, k=10):
        """混合检索"""
        # 语义相似度
        semantic_results = self.embedding_store.search(query, k)
        
        # PPR
        ppr_results = self._personalized_pagerank(query, k)
        
        # 混合评分
        combined = self._combine_scores(
            semantic_results, 
            ppr_results,
            alpha=0.9, 
            beta=0.1
        )
        
        return combined[:k]
```

## 对 neoclaw 的启发

### 图增强 Agent

```python
# neoclaw v2.1.0
class GraphAugmentedAgent:
    """图增强 Agent"""
    
    def __init__(self):
        self.memory = GAAMAMemory()
        self.llm = LLM()
    
    def remember(self, conversation):
        """记住对话"""
        self.memory.add_conversation(conversation)
    
    def recall(self, query):
        """回忆相关信息"""
        return self.memory.retrieve(query)
```

## 总结

**GAAMA 的核心贡献**:
1. **概念介导图谱**: 四节点 + 五边
2. **三步管道**: 分离廉价操作和 LLM 调用
3. **混合检索**: 语义 + PPR
4. **Hub Dampening**: 解决 mega-hub 问题

**对 Project Neo 的意义**:
- claw-mem: 知识图谱 + 混合检索
- neoclaw: 图增强 Agent

## 核心贡献

> "NSA 是一种原生可训练的稀疏注意力机制，结合算法创新与硬件对齐优化实现高效长上下文建模。"

**关键结果**:
- 64K 序列加速: 解码 **11.6×**, 前向 **9.0×**, 反向 **6.0×**
- 性能匹配或超过 Full Attention
- 支持端到端训练

---

## 问题: 现有稀疏注意力的局限

### 推理效率的幻觉

| 问题 | 描述 |
|------|------|
| **阶段限制** | 仅在解码或预填充阶段应用稀疏 |
| **架构不兼容** | 不适配 MQA/GQA 等现代架构 |
| **内存访问** | 计算减少但内存访问仍高 |

### 训练支持缺失

- 大多数方法仅推理时稀疏
- 预训练仍用 Full Attention
- 无法充分利用稀疏模式

### 层次稀疏记忆

```python
# claw-mem v2.1.0
class HierarchicalSparseMemory:
    """层次稀疏记忆"""
    
    def __init__(self):
        # 三路分支
        self.compressed = CompressedMemory()  # 粗粒度
        self.selected = SelectedMemory()      # 细粒度
        self.sliding = SlidingMemory()        # 局部
    
    def retrieve(self, query):
        """三路检索"""
        # 1. 压缩注意力 (全局)
        global_ctx = self.compressed(query)
        
        # 2. 选择注意力 (重要)
        important = self.selected(query)
        
        # 3. 滑动注意力 (局部)
        local = self.sliding(query)
        
        # 门控聚合
        return self._gate(global_ctx, important, local)
    
    def _gate(self, global_ctx, important, local):
        """门控聚合"""
        # 学习门控权重
        g1, g2, g3 = self._compute_gates(
            global_ctx, important, local
        )
        
        return g1 * global_ctx + g2 * important + g3 * local
```

## 对 neoclaw 的启发

### 长上下文 Agent

```python
# neoclaw v2.1.0
class LongContextAgent:
    """长上下文 Agent"""
    
    def __init__(self):
        self.memory = HierarchicalSparseMemory()
        self.llm = LLM()
    
    def process_long_context(self, context):
        """处理长上下文"""
        # 使用 NSA 风格稀疏检索
        relevant = self.memory.retrieve(context.query)
        
        # 高效推理
        return self.llm.generate(context.query, relevant)
```
